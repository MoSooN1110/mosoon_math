# 集合論

## 集合

### 定義 1.1 - 集合   <!-- DEF1.1 -->

**集合(Set)** とは要素(element)と呼ばれる対象の集まりである。

</br>
$A$を集合としたとき、$x$が$A$の要素であることを$x \in A$ と表記する。Aの要素でない場合$x \notin A$と表記する。
